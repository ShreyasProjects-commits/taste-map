# Taste Map

A full-stack restaurant discovery web app that combines multiple data sources
to return filtered, deduplicated restaurant results as map pins and cards.

Built as a CPD project for COMP4060 — Advanced Software Engineering.

## Tech stack

- **Backend**: Node.js, Express
- **Frontend**: React (Vite)
- **Database**: PostgreSQL via Supabase
- **Data sources**: Foursquare Places API, OpenStreetMap (Overpass)
- **Testing**: Jest
- **Deployment**: Render (backend), Netlify (frontend)

## Status

🚧 Under active development — Week 1 of 4.

## Running locally

See `backend/.env.example` for required environment variables.

```bash
cd backend
npm install
npm run dev
```

Backend runs on http://localhost:3000.
